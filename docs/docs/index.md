# Flint - VS Code Extension for Ignition

Flint is a VS Code extension that lets you browse and manage Ignition projects directly in Visual Studio Code.

## What It Does

- **Browse Projects** - View your Ignition project resources in VS Code's explorer panel
- **Manage Resources** - Create, rename, delete, and duplicate resources
- **Search** - Find resources by name or content
- **Merge Conflicts** - Resolve Git merge conflicts in encoded Python scripts with a visual editor
- **Tool Integration** - Open projects in Designer or view backups with Kindling

See [Features](/features) for detailed information on all capabilities.

## Quick Start

1. **Install** the extension from VS Code marketplace
2. **Create** a `flint.config.json` file in your workspace
3. **Configure** your project paths and gateways
4. **Browse** your projects in the Flint Project Browser

See [Getting Started](/getting-started) for detailed setup instructions.
